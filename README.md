# Weather-App
The Weather App is a web-based application developed using HTML, CSS, and JavaScript that fetches real-time weather information using the OpenWeatherMap API.

# Weather App

## Project Overview

The Weather App displays real-time weather information using the OpenWeatherMap API.

Users can search weather conditions by entering a city name.


# Features

- Real-time weather data
- Temperature display
- Humidity information
- Weather condition display


# Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API
# API Key Setup

Get free API key from:

https://openweathermap.org/api

Replace API key in script.js

# Run Project

Open index.html in browser.
